# TikTok Domain Verification

1. Descarga el archivo de verificación desde TikTok Developers.
2. Copia ese archivo (sin modificar nombre ni contenido) dentro de esta carpeta `docs/`.
3. Haz push a `main`.
4. Comprueba que abre en:
   - `https://mrchorusman.github.io/Auto-AAH-RRSS/<nombre-archivo-verificacion>`
5. En TikTok Developers, pulsa **Verify**.
