Términos de Servicio

Última actualización: 2026-03-12

Estos términos regulan el uso del sistema de automatización de contenidos de A Altas Horas para la publicación en redes sociales.

Uso permitido

El servicio se usa para generar borradores, imágenes y copys de episodios del podcast, y publicar contenido en cuentas autorizadas por el propietario.

Propiedad intelectual

Las marcas, logos y contenidos del podcast pertenecen a A Altas Horas y/o sus respectivos titulares.

Limitación de responsabilidad

Se realiza el máximo esfuerzo para publicar contenido correcto, pero no se garantiza disponibilidad continua de APIs de terceros.
